# YouTube Undo/Redo Extension
### A simple Chrome extension that adds undo/redo functionality to YouTube videos.

## Features

Undo/Redo Controls: Quickly return to previous positions in a video

Native Integration: Buttons appear near YouTube's native controls

Seamless Experience: Works with YouTube's dynamic interface

## Installation

Download or clone this repository

Open Chrome and go to chrome://extensions

Enable "Developer mode" (toggle in top-right)

Click "Load unpacked" and select the extension folder

## Usage

Play any YouTube video

Use the undo (↩) and redo (↪) buttons that appear near the video controls

Buttons will be disabled when no actions are available

## Notes
> Works on most YouTube video pages

> History is reset when changing videos or reloading the page

> Requires Chrome or any Chromium-based browser
